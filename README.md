# Pyx's Modern Theme
---

A modern, unibody Discourse theme with a dynamic background.

![Theme Creator screenshot](screenshots/image.png)


---

## Discussion

Please use the [Discourse Meta](https://meta.discourse.org/t/pyxs-modern-theme/357665) thread for concerns, feedback, and nice comments.

## Issues

Issues are welcome using the issues tab. Please make your title be as descriptive as possible, state the issue at hand, and show me visually what is happening that is not intended. I look into all issues when I can, please be patient.

## Contributions

Pull requests for fixes or style updates are also welcome as long as they do not tweak the visual style too much. PRs/issues should be reviewed within a couple of days.
